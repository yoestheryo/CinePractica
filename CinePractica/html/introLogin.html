MANUAL USUARIO

Introducción guía Usuario

Esto es un manual de instrucciones para que los usuarios encargados de utilizar este programa, tengan una guía de uso de las diferentes opciones que se presentan. Al ser un programa de gestión para una cadena de Cine, todo el contenido, será referente a este campo. La función de este manual es ayudar al usuario en caso de no saber cómo realizar una tarea o si le surge algún problema. Este programa consta de una pantalla principal, donde el usuario debe registrarse para acceder al programa y más adelante habrá un menú con las diferentes opciones que iremos explicando paso a paso.

Tipos de Usuarios

Antes de empezar, debemos aclarar que cada usuario de este programa deberá tener asignado su nombre de usuario y su propia contraseña. He de destacar que inicialmente se le han añadió dos tipos de usuario: el ‘administrador’ con su contraseña, y el ‘basico’, con otra contraseña aparte. Estos usuarios se distinguen por sus permisos y derechos de acceso. Esto quiere decir que el usuario ‘administrador’, será un empleado de mayor rango que puede realizar más tareas que el usuario ‘basico’.

Login

Como se muestra en la siguiente imagen, esta es la primera pantalla que nos aparece cuando ejecutamos el programa. Como hemos comentado anteriormente, habrá diferentes perfiles de usuario, cada uno tiene sus propias credenciales. En el primer cuadro de texto, como se indica, se debe introducir el nombre de usuario: ‘administrador’ o ‘basico’.

Si no está seguro de haber escrito bien la contraseña o el nombre de usuario, puede pulsar ‘Cancelar’ o borrar manualmente el texto escrito en los campos de texto. Cuando esté seguro de haber introducido bien los datos, pulse ‘Aceptar’ para proceder a la siguiente ventana. En las siguientes imágenes, se muestra un ejemplo de la apariencia de la ventana tras haber puesto el nombre de usuario y la contraseña de cada uno de los dos usuarios.

Cuando se escribe la contraseña, no será posible visualizarla, ya que está encriptada por motivos de seguridad y aparecerán asteriscos en su lugar. Si resulta que las credenciales son incorrectas, ya se la contraseña o el nombre de usuario, saltará una ventana con el siguiente mensaje:

Debe cerrar esta ventana para poder introducir nuevamente los datos, teniendo cuidado de escribirlos correctamente.

Menú Principal

Cuando se pulse el botón ‘Aceptar’ y las credenciales sea correctas, tendremos delante otra ventana que será el menú principal, compuesto por una barra de menú con cinco pestañas: Asistencias, Personas, Clientes, Cines y Ayuda.

Esta es la pantalla de Menú Principal. Cada una de estas pestañas, al clicar en ellas, despliega otras opciones. Si la persona que se ha registrado es el ‘administrador’, entonces aparecerán cuatro opciones de cada una de estas pestañas. Es decir, las tareas que el ‘administrador’ puede crear son: Altas, Bajas, Modificaciones y Consultas de cada una de las pestañas. Lo que significan que pueden Crear o registrar nuevas Asistencias, Personas, Clientes y Cines y también podrán, Borrar datos, modificarlos o consultarlos. Si el registro se realiza con las credenciales del usuario ‘basico’, solo podrán realizar dos tareas: Alta y Consulta de cada una de las pestañas, es decir, crear o registrar nuevas Asistencias, Personas, Clientes y Cines. Estas serían las dos posibilidades de ventanas dependiendo del perfil de usuario.

Si tiene alguna otra duda, siga consultando el resto de la guía.
